Using the Number Line with the Big Four.
The number line is just a line with numbers marked off at regular intervals.
You probably saw your first number line when you were first figuring out how to count to ten.
In this section, I show you how to use this trusty tool to perform the Big Four operations (adding, subtracting, multiplying, and dividing) on relatively small numbers.
<rus>
Использование числовой линии с большой четверкой.
Цифровая строка - это просто строка с номерами, отмеченными через равные промежутки.
Вы, вероятно, видели свою первую числовую линию, когда вы впервые поняли, как считать до десяти.
В этом разделе я покажу вам, как использовать этот надежный инструмент для выполнения операций большой четверки (сложение, вычитание, умножение и деление) относительно небольших чисел.

<eng>
The number line can be a useful tool for adding and subtracting small numbers:
- When you add, move up the number line, to the right.
- When you subtract, move down the number line, to the left.
To multiply on the number line, start at 0 and count by the first number in the problem as many times as indicated by the second number.
<rus>
Числовая строка может быть полезным инструментом для сложения и вычитания небольших чисел:
- При добавлении переместитесь вверх по числовой строке вправо.
- Когда вычитаете, двигайтесь вниз по числовой линии влево.
Чтобы умножить числовую строку, начните с 0 и посчитайте по первому числу в задаче столько раз, сколько указано вторым числом.

To divide on the number line, first block off a segment of the number line from 0 to the first number in the problem.
Then divide this segment evenly into the number of pieces indicated by the second number.
The length of each piece is the answer to the division.
<rus>
Чтобы разделить числовую линию, сначала заблокируйте отрезок числовой линии от 0 до первого числа в задаче.
Затем разделите этот сегмент равномерно на количество частей, указанное вторым числом.
Длина каждого куска является ответом на деление.
